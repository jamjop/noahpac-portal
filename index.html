About

Noah is a Physician Associate-Certified (PA-C) practicing in outpatient primary care. This site is a personal reference project — fast, offline-capable tools built for frontline providers who need guidelines, calculators, and drug references at the point of care, without logins or paywalls.

All calculations run entirely in your browser. No data is collected or transmitted. Content reflects current published guidelines and validated clinical tools.

Guideline content is regularly updated through automated API integrations and scheduled data pulls from authoritative sources — so recommendations stay current without manual maintenance.

Disclaimer

Not medical advice. These tools summarize published guidelines and validated scoring systems for reference use only. Always apply clinical judgment, read primary sources, and individualize care. Verify calculator results before guiding treatment decisions.
